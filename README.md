## Install deps
`$ npm install`

## Run the server
`$ npm run server`

You will need to create a keys_dev.js in the server config folder with
```js
module.exports = {
  mongoURI: 'YOUR_OWN_MONGO_URI',
  secretOrKey: 'YOUR_OWN_SECRET'
};
```
